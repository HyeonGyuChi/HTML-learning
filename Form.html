    비밀번호 :



    ↑ localhost/login.php로 입력받은 데이터인 id = Hello pwd = html 을 보냄

    input 태그 : 사용자로부터 정보를 입력받기 위해사용
    input type = "?" : ?에따라 사용자로부터 입력받는 종류지정
    text = 글자
    password = 입력된 글자 *표시, 글자수 파악가능
    submit = 입력된 데이터를 서버로 보내기위한 버튼생성

    form 기본속성
    action = "http://~" : 입력한 데이터정보를 http://~로 보냄
    name = 컨트롤의 이름을 정해줌 → 서버에 알맞은 데이터 정보를 전송하기위해 지정

    form 속성
    value = 기본적으로 적혀있는 text 설정
