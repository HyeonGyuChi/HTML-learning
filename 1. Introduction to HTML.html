Review

  1. HTML = HyperTextMarkup Language → 웹사이트 제작에 최적화 HTML + JavaScript or CSS
  2. 대부분 HTML elements는 opening tags와 closing tags로 이루어져있다 → 사이엔 raw text나 다른 HTML tag를 담는다.
  3. HTML은 상속의 특성을 가지고 있다 외부tag = 부모tag / 내부tag = 자식tag → 자식은 부모의 특성을 상속받는다.
  4. 눈에 보이는 대부분 element는 body 에 들어가있다 → 문서의 제목,저자를 기록한 문서의 속성은 head태그에 넣는다.
  5. p span div 태그는 글자를 블록화 시켜준다.
  6. h1~h6를 사용하여 heading (h 뒤의 숫자가 커질수록 소제목)
  7. em과 strong을 사용하여 강조표현
  8. br태그 = LineBreak = 다음행으로 넘김
  9. list는 ol(=ordinary)과 ul(=unordinary)태그가 있고, 자식태그로 li를 품는다.
  10. 이미지를 첨부시 img태그 비디오 첨부시 video태그
    im의 Attrdution으로는 scr와 width height가 있다. (scr = "url") / 닫는Tag필요X
    alt로 이미지로딩 실패시 대체텍스트 표시설정해주기
    video의경우 img와 Attridution이 비슷하지만 닫는Tag가 필요하고 control 속성으로 사용자가 직접조정가능
